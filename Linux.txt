vi和vim的使用：
	一般模式：用 vi 和 vim 命令打开文件（vim test.txt) ,进入一般模式。
		  若没有先创建，再打开。


	编辑模式：在一般模式下，按 i 或者 a ，进入编辑模式；
		  可以编辑，不能保存编辑的内容。
		  按 esc 回到一般模式

	命令模式：在一般模式下，按 shift + : ,进入命令行模式。
			q! ：不保存强制退出编辑器。
			wq : 保存并且退出。
			q  ：只退出。

	快捷键：
		复制当前行：一般模式下，按yy，复制当前行
					按p，粘贴到光标所在的下一行。
		
		复制当前行以及下面几行：一般模式下，按nyy，复制。

		在文本文件中查找关键字：在命令行模式下，输入 / + 关键字，回车。
					按n表示光标查找下一个关键字。
		
		删除光标所在行：在一般模式下，按 dd 。

		删除光标所在的行往下5行：在一般模式下，按 5dd 。

		撤销上次编辑的内容：在一般模式下，按 u 。

		显示行号：命令行模式下，设置文件的行号，set nu;取消文件的行号,set nonu。

用户管理：

	添加用户：useradd 用户名   （有权限的用户来添加）
		设置密码：passwd 用户名

		useradd -d /home/ls lisi：创建一个账号叫lisi，并且给lisi指定家目录/ls。
	
	删除用户级联删除它的主目录：userdel -r 用户名。

	查看用户信息：id 用户名

	切换用户：su 用户名

组管理：
	linux中的组相当于角色的概念，可以对有共性的用户进行统一管理。
	每一个用户至少属于一个组，不能独立于组存在，也可以属于多个组。

	添加组： groupadd 组名

	删除组： groupdel 组名

	把用户添加到组中：gpasswd -a 用户名 组名

	从组中移除用户：gpasswd -d 用户名 组名

	添加用户时，指定主组：useradd -g 组名 用户名

文件和目录操作的命令
	查看当前所在目录：pwd

	查看指定目录下所有的子目录和文件：ls [指定目录]
	[] 代表可选	-l 以列表形式显示（相当于 ll）
			-a 显示隐藏目录

	切换到指定目录：cd  绝对目录（盘符开始）和相对目录（目录名开始的）
			cd ~ 返回到主目录
			cd ..返回到上级目录

	创建目录：mkdir 目录。
	创建多级目录:mkdir -p 目录路径/目录。

	删除空目录：rmdir 目录。

	创建一个或多个空文件：touch 文件名称列表。

	递归复制整个文件夹： cp source（文件名） dest（目标目录）
	递归复制整个文件夹： cp -r source（文件名） dest（目标目录）

	删除文件或目录：rm [可选] 要删除的文件或目录
	递归删除整个文件夹：rm -r 要删除的文件夹
	强制删除不提示：rm -f 要删除的文件或目录

	重命名：mv oldNameFile newNameFile
	移动文件: mv 要移动的文件 目标路径
	
	只读查看文件：cat [可选] 要查看的文件名
	显示行号：-n
	分页显示内容（空格翻页）：cat -n 文件名 |more

	分页显示文本内容：more 文件名
		空格：翻页； 回车：翻一行； q：离开； Ctrl+F：向下滚动一屏；
		Ctrl+B ：返回上一屏； = ：输出当前行的行号； ：+ f  ：输出文件名和当前行的行号 

	分页查看文本内容：less 文件名
		不需要一次将整个文件加载后显示，而是根据显示需要加载内容，对显示大型文件具有较高的效率。
		空格键/pagedown 向下翻一页； pageup 向上翻一页； /字串 向下搜索字串（n向下搜索，N向上搜索）
		？字串 向上搜索字串； q 离开。

	查看文件头10行内容：head 文件
	head -n 5 文件：查看文件头5行内容。
	
	查看文件尾10行内容：tail 文件
	tail -n 5 文件：查看文件尾5行内容

	输出变量或常量内容到控制台：echo $+变量名 / echo 常量名
	
	把查看命令的结果输入给目标文件中（若不存在，则新建;若存在则覆盖。）：查看命令 > 目标文件
	ls > res.txt

	把查看命令的结果输入给目标文件中（若不存在，则新建;若存在则追加。）：查看命令 >> 目标文件

压缩和解压指令
	gzip/gunzip指令
		gzip 文件：将文件压缩为*.gz文件存放在原文件所在目录，压缩成功会把原文件删除，
		用于压缩单个文件。

		gunzip 文件：解压文件，解压成功放在原压缩文件目录，并把原压缩文件删除

	zip/unzip指令
		zip XXX.zip 将要压缩的内容：将指定文件或目录压缩成XXX.zip文件，用于压缩所有
		文件结构。
			-r：递归压缩，即压缩目录。

		unzip xxx.zip：解压内容
			-d 目录：指定解压后文件的存放目录

	tar命令
		tar xxx.tar.gz	:打包或者解压文件
			-c：产生.tar.gz打包文件
			-v：显示详细信息
			-f：指定压缩后的文件名
			-z：打包同时压缩
			-x：解压.tar.gz文件
			-C: 指定解压到哪个目录
		eg：tar -zcvf xxx.tar.gz 打包的文件组（用空格隔开）：打包文件组为xxx.tar.gz
		    tar -zxvf xxx.tar.gz -C 目录 ：将xxx.tar.gz解压到目录下。
	
文件/目录和组
	在linux中，每一个用户至少属于一个组，用户不能独立于组存在，一个用户可以属于多个组。
	
	在linux中，每一个文件或者目录也必须属于一个组，而且只能属于一组。
	在文件或者目录通过组来控制哪些用户可以对它进行那些操作，即文件或者目录的访问权限。
	在文件或者目录看来，linux系统中所有的用户分为三类：
		所有者：默认情况下，是创建者；可以修改
		同组用户：跟文件或者目录属于同一个组的用户
		其他组用户：既不是文件或者目录的所有者，也不是同组用户

	查看文件所有者和所在组命令：ls -ahl 文件名  （a-all,h-human,l-list）

	修改文件所有者命令：chown 新所有者 文件名
		chown newowner:newgroup file 同时修改文件的所有者和所在组。
		-R	：如果是目录则使其下所有子文件和目录递归生效。
	修改文件所在组命令：chgrp 新组名 文件名
		-R	：如果是目录则使其下所有子文件或目录递归生效
	
权限管理
	在liunx中，任何文件或者目录都有三种权限：读(Read),写(Write),执行(Execute).
		对于文件：
			读：可以读取、查看文件的内容，eg：cat,more,less,head,tail等
			写：可以修改文件的内容，eg:vi或者vim等
			执行：如果该文件是可执行文件（.sh），可u直接运行，比如 ./xxx.sh。
		
		对于目录：
			读：可以读取，查看目录下的内容，eg：ls等
			写：可以修改目录中的内容，创建、删除子目录，创建、删除、重命名文件（目录）
			执行：可以进入该目录，eg：cd等。
		
	在linux中，任何文件或者目录都有三部分权限：所有者权限、同组用户权限、其他组用户权限。
	ll 查看权限：2-4为所有者权限，5-7同组权限，9-11同族权限。
	修改权限命令：	chmod u=rex,g=rx,o=x 文件目录名
			说明：	u、g、o、a分别代表文件所有者、文件所在组用户、其它组用户、所有用户
				=、+、-分别代表设置权限、增加权限、去掉权限
			chmod 三个数字 文件名
			说明 r=4,w=2,x=1;
